About This Project

An educational exploration of web fundamentals

Project Overview

This project is a small HTML5-only website created as a primer on semantic web development. It contains four articles covering topics about the history of the web, web security, mobile-first development, and internet statistics in the Middle East and Lebanon. A final page presents the references in IEEE style.
